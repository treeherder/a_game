# a_game Wiki

Welcome to the project wiki for **a_game**, an extensible entity-component system for multiplayer hack-and-slash roguelike arena games.

## Documentation

- [Getting Started](Getting-Started.md) - Installation, quick start, and basic usage
- [API Reference](API.md) - Core API functions and usage patterns
- [Events & Networking](Events.md) - Event system and multiplayer architecture
- [Map Generation](Map-Generation.md) - Procedural labyrinth generation guide

## Key Features

- **Single-root atom architecture** for simple serverless multiplayer
- **Event sourcing** with pure functions and JSON serialization
- **Procedural labyrinth generation** with guaranteed connectivity
- **Entity-Component-System (ECS)** design pattern
- **Comprehensive test coverage** with property-based testing
